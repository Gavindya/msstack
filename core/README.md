## Add as a project dependency
1. Add snapshot version to maven pom file
```xml
<!-- load version with latest commit -->
<dependency>
    <groupId>com.grydtech.msstack</groupId>
    <artifactId>model-converter</artifactId>
    <version>-SNAPSHOT</version>
</dependency>
```
